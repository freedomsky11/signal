# Signal Project

This is the home of the **Signal** tech newsletter.

## Usage

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```
   Open http://localhost:5173 to browse the archive.

3. **Archive Structure**:
   All posts are stored in `docs/posts/YYYY/MM/`.
   Use the `newsletter-archiver` skill to add new posts.
