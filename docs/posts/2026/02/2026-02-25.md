今日的信号汇聚于一个核心：**智能体工程化（Agentic Engineering）的系统性重构**。当 Perplexity 从搜索转向“计算机”编排平台，当 AT&T 通过多智能体堆栈将成本降低 90%，我们正目睹从“模型调用”到“系统构建”的范式转移。然而，这种复杂性也带来了新的隐患：旧时代的 API 权限管理在 LLM 面前失效，模型间的“知识蒸馏”演变为工业级对抗。

## 智能体与系统架构

### [Perplexity 发布 Computer：从搜索引擎到编排平台](https://smol.ai/issues/2026-02-25-wtf-happened)
**摘要**：Perplexity 推出了名为“Computer”的智能体平台，支持多模型路由、按需计费以及异步并行子智能体工作流。与此同时，OpenAI 发布了 GPT-5.3-Codex，旨在提升长程任务的完成速度与成功率。
**点评**：**智能体的竞争已不在于单一模型的参数量，而在于对工作流的“调度能力”。Perplexity 的转型预示着未来的 AI 交互不再是对话框，而是一个拥有多模型调用权限的异步操作系统。**

### [通过 CLI 优化 MCP：将智能体工具成本降低 94%](https://kanyilmaz.me/2026-02-23-cli-vs-mcp.html)
**摘要**：开发者通过将模型上下文协议（MCP）转换为轻量级的 CLI 模式，大幅减少了智能体在初始化阶段加载 JSON Schema 所消耗的 Token。通过按需发现工具细节，而非一次性注入所有工具定义，实现了成本的极致压缩。
**点评**：**这是典型的“工程化回溯”。当模型能力过剩而上下文成本昂贵时，开发者开始借用传统的 CLI 思路来优化 AI 的“说明书”。在 Agentic 时代，协议的简洁程度直接决定了 ROI。**

### [AT&T 的 80 亿 Token 日均处理：重构多智能体堆栈降低 90% 成本](https://venturebeat.com/orchestration/8-billion-tokens-a-day-forced-at-and-t-to-rethink-ai-orchestration-and-cut)
**摘要**：AT&T 首席数据官分享了其企业级 AI 转型经验，通过将任务从大型模型迁移至小语言模型（SLMs）并采用多智能体架构，成功在处理海量流量的同时显著降低了开销。
**点评**：**企业级 AI 的真相是：SLM 负责执行，Agent 负责拆解，LLM 只负责兜底。这种分层架构是解决大模型落地成本难题的唯一出路。**

## 安全、风险与主权

### [Google API 密钥风险：当旧规则遭遇 Gemini](https://trufflesecurity.com/blog/google-api-keys-werent-secrets-but-then-gemini-changed-the-rules)
**摘要**：多年来，Google API 密钥（如 Maps 密钥）被视为非敏感信息并常被嵌入前端。然而，Gemini 允许使用相同的密钥进行身份验证。安全研究人员发现数千个公开密钥可被用于访问受害者的私有 Gemini 数据或刷单扣费。
**点评**：**这是典型的“能力溢出导致的权限漏洞”。模型能力的通用化打破了传统 API 的权限边界，开发者必须重新审视那些原本认为“安全”的公开资产。**

### [Anthropic 控诉“工业级蒸馏攻击”](https://smol.ai/issues/2026-02-23-anthropic-distillation)
**摘要**：Anthropic 声称 DeepSeek、月之暗面（Moonshot）等公司通过数万个欺诈账号，对 Claude 模型进行了超过 1600 万次交互，旨在通过 API 输出提取并蒸馏模型能力。
**点评**：**“蒸馏”已成为后 Scaling Law 时代的工业级军备竞赛。当数据红利耗尽，顶尖模型的输出本身就成了最高质量的训练语料。API 抗滥用技术将成为模型服务商的下一道护城河。**

## 算力经济与硬件趋势

### [Nvidia 第四季度营收突破 681 亿美元，增长 73%](https://www.theverge.com/tech/884915/nvidia-keeps-riding-the-ai-boom-with-q4-revenue-up-73-percent-to-68-1-billion)
**摘要**：Nvidia 财报再次刷新记录，其中数据中心业务贡献了 623 亿美元。这一增长反映了全球范围内对 AI 基础设施的持续重注。
**点评**：**营收数字背后的真正信号是：算力依然是目前最稀缺、最硬的通货。在 AGI 到来之前，这种“算力税”将持续重塑科技巨头的资产负债表。**

### [内存（RAM）占 PC 物料成本比重升至 35%](https://arstechnica.com/gadgets/2026/02/ram-now-represents-35-percent-of-bill-of-materials-for-hp-pcs/)
**摘要**：HP 等厂商的数据显示，受 AI PC 本地运行模型的需求驱动，大容量内存的需求激增，RAM 已成为 PC 硬件中成本占比最高的部件之一。
**点评**：**端侧 AI 的瓶颈不在于 CPU/NPU 的算力，而在于内存带宽与容量。PC 的价值定义正在从“处理核心”转向“数据缓存库”。**

## 开放生态

### [阿里巴巴发布 Qwen 3.5 Medium：挑战顶尖闭源模型性能](https://venturebeat.com/technology/alibabas-new-open-source-qwen3-5-medium-models-offer-sonnet-4-5-performance)
**摘要**：阿里开源了 Qwen 3.5 中型模型，通过 4-bit 权重和 KV 缓存量化技术，在保持极高准确度的同时大幅降低了对硬件的要求，性能直逼闭源顶尖模型。
**点评**：**开源生态的快速迭代正在不断挤压闭源模型的溢价空间。对于大多数开发者而言，Qwen 这类高性能开源模型提供的“本地化部署”能力比订阅制 API 更具吸引力。**
